## Description
 - PopCat app in Command-line interface("CLI")

## How to use
 - (1) Install PopCat - If you choose installer installation
 - (2) Extract Embed Archive - If you choose embed installation
 - (3) Run PopCat
 - (4) Press Ctrl+C(keyboard interrupt) to exit

 - Windows
 - MacOS
 - Linux
